1. Счёт: счётчик увеличивается за каждый найденный сет без 
использования помощи. В конце игры счёт выводится в диалоговом
окне

2. Таймер. Показывает время после начала игры. Выводится 
в диалоговом окне в конце игры

3. Игра вдвоём: 
3.1. При запуске появляется окно с выбором игры одному/вдвоём
3.2. Два счётчика очков
3.3. Выбор активного игрока - тот кому будет засчитано очко
3.4. Итоговое окно: победитель + количество очков

4. Перезапуск игры: в финальном окне добавить кнопку новой 
игры

5. Правила игры:
5.1. Добавить кнопку "Правила"
5.2. Окно с правилами: текст, изображения сетов, 
при необходимости скролл

6. Таблица рекордов:
6.1. Запись время/счёт по итогам игры
6.2. Отображение списка лидеров
